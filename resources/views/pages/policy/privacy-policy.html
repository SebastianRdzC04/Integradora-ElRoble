Política de Privacidad

La privacidad de los usuarios es importante para nosotros. Este proyecto tiene fines educativos y no recopila información sensible ni utiliza los datos con fines comerciales. A continuación, te explicamos cómo manejamos tu información.

1. Datos Recopilados

Solo se utiliza tu correo electrónico para verificar tu identidad y permitir el acceso a la aplicación. Este dato se guarda en nuestra base de datos para registrar el inicio de sesión.

2. Uso de los Datos

El correo electrónico proporcionado se guarda únicamente para registrar el inicio de sesión. No se comparte ni se utiliza con ningún otro propósito.

3. Seguridad

Tomamos medidas básicas para proteger la información, pero ten en cuenta que este proyecto no maneja datos sensibles ni utiliza medidas de seguridad avanzadas.

4. No Acceso a Datos Sensibles

No recopilamos ni almacenamos datos sensibles como contraseñas o información personal más allá del correo electrónico.

Este proyecto es una aplicación educativa destinada a practicar el trabajo de programación, y no tiene fines comerciales ni de recolección de datos más allá de lo necesario para su funcionamiento.
